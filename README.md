# 🎵 本地音乐播放器（Qt C++）

基于 Qt 6 开发的简洁本地音乐播放器，深色主题，功能完整。

---

## 项目结构

```
MusicPlayer/
├── MusicPlayer.pro     # qmake 工程文件
├── main.cpp            # 程序入口
├── mainwindow.h/cpp    # 主窗口（播放器 UI + 逻辑）
├── playlist.h/cpp      # 播放列表模型（QAbstractListModel）
└── trackitem.h         # 歌曲数据结构
```

---

## 环境要求

| 项目 | 要求 |
|------|------|
| Qt 版本 | Qt 6.2+ |
| 必须模块 | `QtMultimedia`、`QtWidgets` |
| 编译器 | MSVC 2019+、MinGW 11+、GCC 10+、Clang 12+ |
| C++ 标准 | C++17 |

> ⚠️ **注意**：在 Windows 上使用 Qt 6 Multimedia 播放 MP3 需要安装  
> **Windows Media Pack** 或确保系统安装了 Media Foundation 相关组件。

## 编译步骤

### 方式一：命令行（qmake）

```bash
cd MusicPlayer
qmake MusicPlayer.pro
make        # Linux/macOS
# 或
nmake       # Windows (MSVC)
mingw32-make  # Windows (MinGW)
```

### 方式二：Qt Creator

1. 打开 Qt Creator
2. 文件 → 打开文件或项目 → 选择 `MusicPlayer.pro`
3. 配置 Qt 6 Kit
4. 点击 **构建** → **运行**

## 功能特性

### 播放控制
- ▶ / ⏸ 播放 / 暂停
- ⏹ 停止
- ⏮ / ⏭ 上一首 / 下一首
- 进度条拖拽跳转
- 音量调节滑块

### 播放模式
- **⇀ 随机播放**：随机顺序播放列表中的歌曲
- **↻ 单曲循环**：循环播放当前歌曲

### 文件管理
- 点击「＋ 添加文件」：选择多个音频文件
- 点击「📁 添加文件夹」：递归扫描整个文件夹
- 直接**拖拽**文件/文件夹到窗口

### 支持格式
`mp3` · `flac` · `wav` · `ogg` · `aac` · `m4a` · `wma` · `opus` · `ape`

### 播放列表
- 双击行切换歌曲
- 显示歌曲编号、标题、艺术家、时长
- 移除选中 / 清空全部

### 键盘快捷键
| 按键 | 功能 |
|------|------|
| `Space` | 播放/暂停 |
| `←` | 后退 5 秒 |
| `→` | 前进 5 秒 |
| `↑` | 音量 +5 |
| `↓` | 音量 -5 |
| `N` | 下一首 |
| `P` | 上一首 |
